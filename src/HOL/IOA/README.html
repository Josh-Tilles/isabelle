IOA: A basic formalization of I/O automata in HOL

Author: Konrad Slind and Tobias Nipkow
Copyright 1995 Technische Universität München

This directory contains a formalization of the meta theory of I/O automata in HOL. This formalization has been significantly changed and extended. The new version is available in the subdirectory HOLCF/IOA. There are also the proofs of two communication protocols which formerly have been here.
