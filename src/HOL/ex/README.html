ex--Miscellaneous Examples

This directory presents a number of small examples, illustrating various features of Isabelle/HOL.

  • Files cla.ML demonstrates the power of Isabelle's classical reasoner.
  • Files meson.ML and mesontest.ML present an implementation of the Model Elimination (ME) proof procedure, which is even more powerful than the classical reasoner but not generic.
  • InSort and Qsort are correctness proofs for sorting functions.
  • Primes is a theory of the divides relation, the greatest common divisor and Euclid's algorithm.
  • NatSum demonstrates the power of permutative rewriting. Well-known identities about summations are proved using just induction and rewriting.
  • MT is a preliminary version of Jacob Frost's coinduction example. The full version is on the directory ZF/Coind.
